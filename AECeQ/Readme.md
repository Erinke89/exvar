## Functions

| Function  | Role| Input | Output | 
| ------------- | ------------- | ------------- | ------------- |

| variant()  |   variants calling  | Bam   | vcf  |
| eQTL()  |   eQTL analysis  | Bam   |     |
| ECeQ()  |   all analysis  | Fastq   | VECeQ interface  |
